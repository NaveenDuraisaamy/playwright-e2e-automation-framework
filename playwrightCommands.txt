Playwright Hooks
----------------
beforeEach: This hook is executed beforeEach individual test.
afterEach: This hook is executed afterEach individual test.

beforeAll: This hook is executed one before any of the tests start running.
afterAll: This hook is executed once afetr all the tests have been run.

Group test
----------
test.describe{

    test()
}
test.describe{
    test()
    test()
